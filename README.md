Wanderlust is a full-stack travel listing web application inspired by Airbnb.  
The project allows users to explore, add, edit, and manage travel property listings through a clean and responsive interface.

This project was built as part of my learning journey in full-stack web development, focusing on frontend design, backend development, and database integration.

---

##  Features

- Browse travel property listings  
- Add new property listings  
- Edit and delete existing listings  
- Responsive design for desktop and mobile  
- Smooth frontend and backend integration

---

## Application Flow (High-Level Design)

1. The frontend sends HTTP requests to backend APIs.
2. Express.js routes handle requests and business logic.
3. MongoDB Atlas stores and retrieves listing data.
4. CRUD operations are performed for property listings.
5. The frontend updates dynamically using JavaScript.
